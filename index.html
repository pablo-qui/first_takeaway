This page explains the main features of the shiny app in the repository.
There are five tabs in the app.

  • The first one is just a brief description of the dataset.
  • The second tab shows the data, allowing the user to visualize and filter the data according to the two categorical variables.
  • The third one allows to represent one of the continuous variables against other and compare the points according to the two categorical variables. It also shows the different means for each variable depending on the categorical variable chosen.
  • The fourth one shows a knn regression model that tries to predict the species of the crab based on the other variables. The user can change the number of neighbours to consider and the train/test size and generate a report with the results.
  • The last one includes the references used for building the app.
